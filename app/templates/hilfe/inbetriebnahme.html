Inbetriebnahme-Anleitung

Zur Hilfe
Diese Anleitung führt Sie Schritt für Schritt durch die Ersteinrichtung von AgroBetrieb. Folgen Sie der Reihenfolge – manche Schritte bauen aufeinander auf.
{# ── Schritt 1: Passwort ändern ── #}

Das erste was Sie nach dem ersten Login tun sollten: Das Standard-Passwort ändern.

Das Standard-Passwort admin123 ist öffentlich bekannt – ändern Sie es sofort!
  1. Klicken Sie oben rechts auf Ihren Benutzernamen
  2. Wählen Sie Passwort ändern
  3. Geben Sie das alte Passwort (admin123) und ein neues sicheres Passwort ein
  4. Mindestens 12 Zeichen, Kombination aus Buchstaben, Zahlen und Sonderzeichen empfohlen
Jetzt Passwort ändern
{# ── Schritt 2: Betriebsdaten ── #}

Geben Sie die Stammdaten Ihres Betriebs ein. Diese erscheinen auf Rechnungen, Berichten und im Briefkopf.

  • Betriebsname – vollständiger Name des Betriebs
  • Adresse – Straße, PLZ, Ort
  • Land – bestimmt MwSt-Satz und Währung (AT = 20% / €, DE = 19% / €, CH = 8.1% / CHF)
  • UID-Nummer / Steuernummer – für Rechnungen erforderlich
  • Telefon / E-Mail / Website – optional, erscheinen im Briefkopf
Betriebsdaten öffnen
{# ── Schritt 3: Module freischalten ── #}

Je nach Betrieb benötigen Sie unterschiedliche Module. Schalten Sie nur die Module frei, die Sie tatsächlich nutzen.

ModulFür wenWas es kann
Legehennen Geflügelbetriebe Herden, Legeleistung, TAMG-Journal, Sortierergebnisse
Milchvieh Rinderbetriebe Rinder, Milchleistung, TAMG-Journal, Tierarztbesuche
Ackerbau Ackerbaubetriebe Schlagkartei, Spritztagebuch (gesetzlich verpflichtend!), Düngerplaner
Ackerbau Pro Ackerbau mit NAPV-Pflicht NAPV-Düngebedarfsermittlung, Bodenuntersuchung, 170 kg N/ha Bilanz, Rote Gebiete
Basis-Module (Buchhaltung, Maschinen, Lager, Fakturierung) sind immer verfügbar und müssen nicht freigeschaltet werden.
Module verwalten
{# ── Schritt 4: Weitere Admins / Personen anlegen ── #}

Legen Sie alle Personen an, die mit AgroBetrieb arbeiten sollen.

Empfohlene Reihenfolge:
  1. Weitere Betriebsadmins anlegen (Stellvertreter, Backup-Admin)
  2. Mitglieder anlegen (alle Personen mit vollem Zugriff)
  3. Buchhaltung anlegen (falls externe Buchhaltung)
  4. Tierärzte anlegen (für TAMG-Journal und Signaturen)
  5. Weitere Rollen nach Bedarf
Beim Anlegen eines Benutzers wird ein temporäres Passwort gesetzt. Das Flag Passwort muss geändert werden erzwingt eine Passwortänderung beim ersten Login.
Benutzerverwaltung öffnen
{# ── Schritt 5: Kontenplan einrichten ── #}

Der Kontenplan bildet die Grundlage für die gesamte Buchhaltung.

  1. Unter Buchhaltung → Kontenplan öffnen
  2. Button Standard-Konten anlegen klicken – erstellt einen vollständigen Basiskontenplan
  3. Eigene Kostenstellen (Klasse 4–8) nach Bedarf ergänzen
  4. Das Bankkonto als Importkonto markieren (wichtig für Bank-Import)
Den Kontenplan vor den ersten Buchungen einrichten – nachträgliche Änderungen sind aufwändiger.
Kontenplan öffnen
{# ── Schritt 6: Bank-Import konfigurieren ── #}

Falls Sie Kontoauszüge als CSV (z.B. ELBA-Export) importieren möchten:

  1. Unter Buchhaltung → Bank-Import → Konfiguration öffnen
  2. CSV-Format Ihrer Bank einstellen (Spalten, Trennzeichen, Datumsformat)
  3. Eine Test-CSV importieren und prüfen ob die Beträge korrekt erkannt werden
  4. Buchungsschlüssel anlegen für häufige Lieferanten/Empfänger
Bank-Import konfigurieren
{# ── Schritt 7: Erste Buchungen / Eröffnungsbilanz ── #}

Falls Sie mit einem bestehenden Kontostand starten, geben Sie Eröffnungsbuchungen ein:

  • Buchungsart Eröffnung wählen
  • Anfangsstand Bankkonto, Kasse, Forderungen, Verbindlichkeiten eingeben
  • Gegenkonto: Eröffnungsbilanzkonto (Klasse 9)
Neue Buchung
{# ── Schritt 8: Ackerbau-Ersteinrichtung ── #}

Falls das Ackerbau-Modul freigeschaltet ist, diese Ersteinrichtung durchführen:

  1. Unter Ackerbau → Schlagkartei alle Schläge anlegen (Name, Fläche in ha, Feldstück-Nr.)
  2. Je Schlag die aktuelle Kultur eintragen (EPPO-Code, Aussaatdatum)
  3. Unter Ackerbau → Spritzmittel die verwendeten Pflanzenschutzmittel anlegen (Stammdaten)
  4. Laufende Spritzungen und Düngegaben im Spritztagebuch bzw. Düngerplaner erfassen
{% if hat_modul_zugriff(current_user, 'ackerbau_pro') %}
Ackerbau Pro: Bodenuntersuchungsergebnisse je Schlag erfassen → bildet die Basis für die NAPV-Düngebedarfsermittlung. Tierbestand für 170 kg N/ha Bilanz anlegen (Daten aus Milchvieh/Legehennen-Modul können übernommen werden).
{% endif %}
Das Spritztagebuch ist gesetzlich verpflichtend – alle Pflanzenschutzmaßnahmen müssen dokumentiert werden (§ 24 Pflanzenschutzmittelgesetz).
{% if hat_modul_zugriff(current_user, 'ackerbau') %} Schlagkartei öffnen {% endif %}
{# ── Schritt 9: Datensicherung ── #}

Nach der Einrichtung sofort ein erstes Backup erstellen.

  • Unter Datensicherung ein vollständiges Backup herunterladen
  • Backup an einem sicheren Ort speichern (nicht nur auf dem NAS)
  • Regelmäßige Backups einplanen (wöchentlich empfohlen)
Datensicherung öffnen
